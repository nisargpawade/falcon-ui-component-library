List

Lists are used to show list items.

Simple List

Use class list-container for simple list given below.

    List Header

  • Item 1
  • Item 2
  • Item 3

              <ul class="list-container p-md">
              <p class="p-lg fw-semi-bold">List Header</p>
                <li> Item 1</li>
                <li> Item 2</li>
                <li> Item 3</li>
              </ul>

Stacked List
