Badge

The Avatar component is used to represent a user, and displays the profile picture, initials.

Badge on Icon

Use class badge for badges or add class badge-l for larger badge.

4

8


          <div class="badge"><i class="fas fa-bell"></i><p>4</p></div>
          <div class="badge badge-l"><i class="fas fa-home"></i><p>8</p></div>

Badge on Avatar

Use class avatar-l-badge avatar-m-badge avatar-badge avatar-s-badge with badge class and avatar-badge-on avatar-badge-off to show status.

Remember to add <p></p> tag for the badge. 


<div class="badge avatar-l-badge avatar-badge-on"><img class="avatar avatar-l" src="..." alt=".."/><p></p>
</div>
<div class="badge avatar-m-badge avatar-badge-off"><img class="avatar avatar-m" src="..." alt=".."/><p></p>
</div>
<div class="badge avatar-badge avatar-badge-on"><img class="avatar" src="..." alt=".."/><p></p>
</div>
<div class="badge avatar-s-badge avatar-badge-off"><img class="avatar avatar-s" src="..." alt=".."/><p></p>
</div>
