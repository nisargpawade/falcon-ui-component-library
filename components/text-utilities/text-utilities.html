Text Utilities

Use Text Utilities to present your content clearly and efficiently.

Falcon UI uses Montserrat font.

Headings

You can directly use the elements h1, h2, h3, h4

This is a Heading

This is a Heading

This is a Heading

This is a Heading


                <h1>This is a Heading</h1>
                <h2>This is a Heading</h2>
                <h3>This is a Heading</h3>
                <h4>This is a Heading</h4>

Text

You can use class p-md for font-size of p element, for other sizes, use class p-lg p-md p-sm

This is a Text

This is a Text

This is a Text

This is a Text

This is small 

            <p class="p-lg">This is a Text</p>
            <p class="p-md">This is a Text</p>
            <p>This is a Text</p>
            <p class="p-sm">This is a Text</p>
            <small>This is small</small>

Font weight

Use class fw-regular fw-semi-bold fw-bold for various font weights.

REGULAR 400

SEMI BOLD 500

BOLD 600


            <p class="p-lg fw-regular">REGULAR 400</p>
            <p class="p-lg fw-semi-bold">SEMI BOLD 500</p>
            <p class="p-lg fw-bold">BOLD 600</p>

Gray Text, Center Text and Right Text

Use class txt-gray txt-center txt-right for below examples.
