Button

Buttons allow users to take various actions.

Simple Button

Use class btn for simple button or add class btn-success btn-warning btn-error btn-info for different colors. 


            <button class="btn">Simple Button</button>
            <button class="btn btn-success">Simple Button 2</button>
            <button class="btn btn-warning">Simple Button 4</button>
            <button class="btn btn-error">Simple Button 5</button>
            <button class="btn btn-info">Simple Button 6</button>
            <button class="btn btn-disable" disabled="disabled">Disabled Button</button>

Outline Button

Add class btn-ol for outline button or add class btn-success-ol btn-warning-ol btn-error-ol btn-info-ol for different colors. 


            <button class="btn btn-ol">Primary</button>
            <button class="btn btn-success-ol">Success</button>
            <button class="btn btn-warning-ol">Warning</button>
            <button class="btn btn-error-ol">Error</button>
            <button class="btn btn-info-ol">Info</button>

Link Button

Add class btn btn-link to make links into buttons and can be used with btn-success btn-warning btn-error btn-info for different colors.

Link Button Link Button Link Button Link Button 

            <a class="btn btn-link" href="#">Link Button</a>
            <a class="btn btn-link btn-success" href="#">Link Button</a>
            <a class="btn btn-link btn-warning" href="#">Link Button</a>
            <a class="btn btn-link btn-error" href="#">Link Button</a>

Icon Button

Use simple button classes and simply add icon <i></i> in the buttons. 


        <button class="btn"><i class="fas fa-shopping-cart"></i>Add to cart</button>
        <button class="btn btn-success"><i class="fas fa-save"></i>Save</button>
        <button class="btn btn-warning"><i class="fas fa-cog"></i>Settings</button>
        <button class="btn btn-error"><i class="fas fa-trash"></i>Delete</button>
