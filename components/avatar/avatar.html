Avatar

The Avatar component is used to represent a user, and displays the profile picture, initials.

Avatar

Use class avatar-l avatar-m avatar-s for avatars of different sizes or avatar for standard avatar size. 


          <img class="avatar-l" src=".." alt="user image"/>
          <img class="avatar-m" src=".."  alt="user image"/>
          <img class="avatar" src=".."  alt="user image"/>
          <img class="avatar-s" src=".."  alt="user image"/>

Square Avatar

Use class avatar-sq-l avatar-sq-m avatar-sq-s for square avatars of different sizes or avatar-sq for standard avatar size. 


          <img class="avatar-sq-l" src=".." alt="user image"/>
          <img class="avatar-sq-m" src=".."  alt="user image"/>
          <img class="avatar-sq" src=".."  alt="user image"/>
          <img class="avatar-sq-s" src=".."  alt="user image"/>
