Avatar

The Avatar component is used to represent a user, and displays the profile picture, initials.

Avatar

avatar are basically used as Profile Pictures. You can use class avatar to use them 


          <img class="avatar-l" src=".." alt=".."/>
          <img class="avatar-m" src=".."  alt=".."/>
          <img class="avatar" src=".."  alt=".."/>
          <img class="avatar-s" src=".."  alt=".."/>

Square Avatar

Use class avatar-square to have square avatar you can also used different Sizes with it 


          <img class="avatar-sq-l" src=".." alt=".."/>
          <img class="avatar-sq-m" src=".."  alt=".."/>
          <img class="avatar-sq" src=".."  alt=".."/>
          <img class="avatar-sq-s" src=".."  alt=".."/>
