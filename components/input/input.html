Input

Text fields allow user input.

Textbox

Use class input-simple for simple textbox to input your data. 


            <input class="input-simple" placeholder="Input Area"></input>

Error success inputs

Add class input-success input-warning input-error along with input-info-success input-info-warning input-info-error for below output.

Email

email is correct

Phone No.

phone no. must have 10 characters

Password

password is wrong


            <div class="margin-b-1">
                <p>Email</p>
                <input class="input-simple input-success" placeholder="..."></input>
                <p class="p-sm input-info-success">email is correct</p>
            </div>
            <div class="margin-b-1">
                <p>Phone No.</p>
                <input class="input-simple input-warning" placeholder="..."></input>
                <p class="p-sm input-info-warning">phone no. must have 10 characters</p>
            </div>
            <div class="margin-b-1">
                <p>Password</p>
                <input class="input-simple input-error" placeholder="..."></input>
                <p class="p-sm input-info-error">password is wrong</p>
            </div>
